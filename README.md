🛒 Best Buy Supermarket

Best Buy is a simple supermarket web application that allows users to browse products and manage accounts.

📌 Features

User registration and login

View all products

Add products to cart

Secure authentication

Responsive design

🛠️ Built With

Node.js

Express.js

MongoDB

Mongoose

Tailwind CSS

📂 Project Structure
Best-Buy/
│── models/
│── routes/
│── controllers/
│── public/
│── server.js
│── package.json
🚀 Installation

Clone the repository:
```
git clone https://github.com/yourusername/best-buy.git
```
Install dependencies:
```
npm install

Create a .env file and add:

MONGO_URI=your_mongodb_connection_string
PORT=5000
```
Start the server:

npm start

Server will run on:

http://localhost:5000
📦 API Example
Get All Users
GET /api/users
Create User
POST /api/users
